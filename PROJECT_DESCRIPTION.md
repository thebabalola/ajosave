# Ajosave - Project Description

## One-Line Description

**Ajosave** is a decentralized community savings platform that brings traditional Ajo/Esusu rotating savings and credit associations (ROSCAs) on-chain with blockchain transparency and automation.

## Short Description

Ajosave digitizes traditional community savings practices using blockchain technology. Built for African communities and accessible globally, it enables trustless, transparent, and automated savings group management through smart contracts on Celo and Base networks.

## Key Features

- **Three Pool Types**: Rotational savings (traditional Ajo/Esusu), target-based savings, and flexible deposits/withdrawals
- **Smart Contract Escrow**: Funds locked in audited, transparent contracts with no custodial risks
- **Multi-Chain Support**: Deployed on Celo (mobile-friendly) and Base (low-cost L2)
- **Automated Enforcement**: Rules, penalties, and payouts execute automatically
- **Modern Web Interface**: Responsive design with wallet integration and real-time tracking
- **Yield Integration**: Optional DeFi yield generation for idle funds

## Problem Statement

Traditional savings groups (Ajo, Esusu, Tontine) face challenges:
- Trust issues between members
- Manual record-keeping and errors
- Geographic limitations
- Lack of transparency
- No automated enforcement of rules
- Risk of fund mismanagement

## Solution

Ajosave solves these problems by:
- **Blockchain Transparency**: All transactions are visible on-chain
- **Smart Contract Automation**: Rules enforced automatically by code
- **Global Accessibility**: Anyone with internet and a wallet can participate
- **Non-Custodial**: Users always control their funds
- **Transparent Tracking**: Real-time dashboard and activity feeds
- **Multi-Chain**: Accessible on affordable, fast networks

## Technical Highlights

- **Smart Contracts**: Solidity 0.8.30 with OpenZeppelin libraries
- **Frontend**: Next.js 15 with TypeScript, Tailwind CSS, shadcn/ui
- **Web3**: Wagmi + Viem for Ethereum interactions
- **Database**: Supabase (PostgreSQL) for off-chain data
- **Networks**: Celo Sepolia & Base Sepolia (testnets deployed)
- **Verified Contracts**: All contracts verified on Sourcify
